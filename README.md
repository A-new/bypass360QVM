# bypass360QVM

添加图标以及版本信息，实现自动化bypass360QVM
主要逻辑参考了大佬T4y1oR的https://github.com/T4y1oR/RingQ/tree/main/QVM250

**改进：**
1、减小了生成的图标的扰动，基本肉眼分辨不出区别（方便钓鱼用）

2、添加的随机版本信息中文件描述、公司名称、文件名等有一定可读性

**使用方法：**

```bat
python 360QVM.py <exe文件> <生成数量>
```
